TEACHER DASHBOARD PROJECT SUMMARY
=====================================

PROJECT OVERVIEW
----------------
The Teacher Dashboard is a comprehensive career exploration and analytics platform built with React, TypeScript, and Node.js. It's designed to help teachers and students explore different career paths while providing detailed analytics on user interactions and behavior patterns.

TECHNOLOGY STACK
----------------
Frontend:
- React 18.3.1 with TypeScript 5.5.3
- Vite 5.4.2 as build tool
- Tailwind CSS 3.4.1 for styling
- Lucide React for icons

Backend:
- Node.js with Express 5.1.0
- MySQL 8.0+ database integration
- MySQL2 driver for database operations

Development Tools:
- ESLint for code quality
- PostCSS and Autoprefixer
- Concurrently for running multiple processes

CORE FEATURES
-------------
1. Career Exploration Dashboard
   - Occupation search and filtering
   - Detailed career information display

   - Role-based content (Teacher, CEO, IT Specialist, etc.)

2. User Analytics & Tracking

   - Session analytics and user journey mapping
   - AI-powered behavior pattern recognition

3. Attendance Management
   - Student attendance tracking
   - Attendance analytics and reporting
   - Integration with main dashboard

4. Course Selection System
   - Course browsing and selection

   - Course management interface

5. AI-Powered Insights
   - ChatGPT integration for advice
   - Behavior analysis and recommendations
   - Engagement scoring algorithms
   - Personalized user insights

DATABASE ARCHITECTURE
--------------------
The system uses MySQL with three main tables:

1. user_interactions
   - Tracks all user interactions (clicks, scrolls, form submissions)
   - Stores interaction metadata, timestamps, and user context
   - Indexed for fast query performance

2. user_sessions
   - Manages user session data
   - Tracks session duration and interaction counts
   - Stores session-specific information

3. user_analytics
   - Aggregated user behavior data
   - Engagement scores and behavior patterns
   - AI-generated insights and recommendations

Additional tables for:
- Attendance tracking
- Advice conversations (ChatGPT interactions)
- Advice analytics
